사용법

📂 파일 업로드 순서

  1. BOM 파일 (.xlsx) 업로드
  2. 좌표 데이터 파일 (.xlsx) 업로드
  3. 각 파일의 시트 선택
  4. 헤더 행 설정 확인 (보통 1행)
  5. 열 이름 설정 — Location, 품명, 규격, Maker, 수량
  6. ▶ 매칭 시작 클릭

🎨 결과 색상 의미

  • 매칭 — BOM·좌표 양쪽 존재, 규격 일치
  • 규격 불일치 — 존재하지만 사이즈·피치 다름
  • BOM만 — BOM에 있으나 좌표 데이터 없음
  • 좌표만 — 좌표 데이터에 있으나 BOM 없음
  • BOM 중복 — 동일 RefDes가 여러 BOM 행에 등장

⚠️ 자동 제외 항목

  • FIDUCIAL — 파이듀셜 마크
  • HOLE — 마운팅 홀 (H1~Hn)
  • 숫자 없는 토큰 — BOTTOM, BOX-CARTON 등
  • FOR 패턴 — FOR CP803S 형태 → 부속품으로 분류
  • 대체품 — Location이 "대체품"인 행
  • 품명 제외 키워드 — SILICONE-RTV, LABEL 등 (설정 가능)

📐 규격 비교 항목

  • 칩 사이즈 — 1005, 1608, 2012, 3216 등
  • 피치 — P5.0, P7.5 등 P값
  • 다이오드 패키지 — SOD / SOT / SMA / SMB / SMC
  • 수량 — BOM 단위수량 vs Location의 RefDes 개수

✏️ 수정 페이지

  • 우측 하단 ✏️ 수정 페이지 버튼으로 진입
  • BOM only 삭제, 중복 제거, 인라인 셀 수정 가능
  • 엑셀 다운로드 시 원본 BOM 서식 유지 + 검증결과 컬럼 추가

📊 BOM 버전 비교 사용법

  1. 상단 탭에서 📊 BOM 버전 비교 클릭
  2. 구버전 BOM과 신버전 BOM 파일 각각 업로드
  3. 두 파일 업로드 완료 시 감지된 열 목록 자동 표시
  4. RefDes 열 이름 확인 (기본값: Location)
  5. 비교에서 제외할 열이 있으면 쉼표로 구분하여 입력
    예: 단계, Maker
  6. 🔍 버전 비교 실행 클릭

비교 결과 항목:

  • ➕ 추가 — 신버전에만 있는 RefDes
  • ➖ 삭제 — 구버전에만 있는 RefDes
  • 🔄 변경 — 같은 RefDes이지만 열 값이 달라진 항목
